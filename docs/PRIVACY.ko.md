# 개인정보 처리방침

최종 업데이트: 2026년 7월 21일

ComfyUI-Nexus는 사용자가 제어하는 ComfyUI 실행 환경을 설정하고 운영하기 위한
로컬 데스크톱 애플리케이션입니다.

## 로컬 데이터

ComfyUI-Nexus는 설정, 실행 환경 파일, 로그, 캐시, 백업, 워크플로,
생성 결과물 및 관련 메타데이터를 사용자의 기기에 저장합니다.<br>
사용 방식에 따라 이 로컬 데이터에는 파일 경로, 프롬프트, 워크플로 내용, 생성 이미지가 포함될 수 있습니다.

앱은 사용자 인터페이스에 정보를 표시하기 위해 로컬 CPU 및 GPU 상태를 처리합니다.<br>
Nexus는 개발자 원격 측정, 광고 또는 분석 서비스를 운영하지 않으며, 이러한 로컬 데이터를 개발자에게 자동 전송하지 않습니다.

## 외부 서비스

사용자가 외부 리소스의 설치, 업데이트, 다운로드 또는 연결을 선택하면 ComfyUI-Nexus는<br>
GitHub, Hugging Face, ComfyUI 저장소 또는 사용자가 설정한 엔드포인트와 직접 통신할 수 있습니다.<br>
이 서비스들은 자체 개인정보 처리방침에 따라 사용자의 IP 주소와 일반적인 요청 메타데이터를 수신할 수 있습니다.

ComfyUI-Nexus는 사용자의 이미지, 프롬프트, 워크플로, 모델 파일 또는 로컬 로그를 Nexus 개발자에게 업로드하지 않습니다.

## 계정과 공유

ComfyUI-Nexus는 Nexus 계정을 요구하지 않습니다. 또한 제3자 광고,
개발자가 운영하는 분석 또는 소셜 공유 기능을 포함하지 않습니다.

## 사용자 제어

사용자는 ComfyUI-Nexus가 사용하는 로컬 파일과 외부 경로를 제어합니다.
사용자가 선택한 외부 ComfyUI 설치본, 모델 라이브러리 및 파일은
사용자의 관리 대상이며 앱 밖에서도 관리할 수 있습니다.
