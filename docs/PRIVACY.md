# Privacy Policy

Last updated: July 21, 2026

ComfyUI-Nexus is a local desktop application for setting up and operating a
user-controlled ComfyUI runtime.

## Local Data

ComfyUI-Nexus stores its settings, runtime files, logs, caches, backups,
workflows, generated outputs, and related metadata on the user's device.<br>
Depending on how the app is used, this local data may include file paths,
prompts, workflow content, and generated images.

The app processes local CPU and GPU status only to display information in the user interface.<br>
Nexus does not operate a developer telemetry, advertising, or analytics service, and it does not automatically send this local data to the developer.

## External Services

When the user chooses to install, update, download, or connect to external resources,<br>
ComfyUI-Nexus may communicate directly with services such as GitHub, Hugging Face, ComfyUI repositories, or user-configured endpoints.<br>
Those services may receive the user's IP address and standard request metadata
under their own privacy policies.

ComfyUI-Nexus does not upload the user's images, prompts, workflows, model
files, or local logs to the Nexus developer.

## Accounts And Sharing

ComfyUI-Nexus does not require a Nexus account.<br>
It does not include third-party advertising, developer-operated analytics, or social sharing features.

## User Control

Users control the local files and external paths used by ComfyUI-Nexus.<br>
External ComfyUI installations, model libraries, and files selected by the user
remain under the user's control and may be managed outside the app.

## Contact

For privacy questions, open an issue in the
[ComfyUI-Nexus GitHub repository](https://github.com/wenovaX/ComfyUI-Nexus/issues).
