# 프로젝트 정보

[English](PROJECTINFO.md)

## 제품명

- 공식 제품명: **Nexus for ComfyUI**
- 짧은 브랜드명: **Nexus**
- 내부/프로젝트 별칭: **ComfyUI Nexus**
- 저장소 이름: **ComfyUI-Nexus**

스토어 목록, 스크린샷, 릴리스 노트, 사용자 문서에는 **Nexus for ComfyUI**를 사용합니다.<br>
**ComfyUI Nexus**는 이 저장소를 가리키는 맥락이 명확한 내부 또는 비공식 프로젝트 별칭으로만 사용합니다.

## 제품 포지셔닝

Nexus for ComfyUI는 ComfyUI를 위한 독립형 데스크톱 컴패니언입니다.

Windows에서 로컬 ComfyUI 작업 공간을 설치, 실행, 모니터링, 복구, 유지 관리할 수 있도록 돕습니다.<br>
이 앱은 컴패니언 및 런타임 관리자이며, ComfyUI의 공식 배포판이 아닙니다.

## 공개 설명

권장 짧은 설명:

> ComfyUI를 위한 독립형 데스크톱 컴패니언.

권장 긴 설명:

> 안내형 설정 흐름, 관리형 확장, 외부 모델 라이브러리 지원, 서버 진단을 통해 Windows의 로컬 ComfyUI 작업 공간을 설치, 실행, 모니터링, 유지 관리합니다.

## 스토어 및 브랜딩 규칙

- 앱을 ComfyUI의 공식 제품처럼 소개하지 않습니다.
- 스토어용 문구에는 **ComfyUI Nexus**보다 **Nexus for ComfyUI**를 우선 사용합니다.
- 이후 패키징 작업에서 명시적으로 이름을 변경하기 전까지 저장소와 기술 식별자는 **ComfyUI-Nexus**로 유지합니다.
- 현재 릴리스가 NVIDIA CUDA GPU를 사용하는 Windows를 대상으로 한다는 점을 알립니다.
- 앱이 독립형 이미지 생성 서비스라는 인상을 주는 "Stable Diffusion Generator" 같은 일반 이름은 피합니다.

## 런타임 소유권

- Vanguard 모드는 Nexus가 소유하는 로컬 ComfyUI 런타임을 관리합니다.
- Architect 모드는 사용자가 관리하는 기존 ComfyUI 작업 공간에 연결합니다.
- 모델, 워크플로, 입력, 출력, 커스텀 노드, 외부 모델 경로 같은 사용자 데이터는 런타임 데이터로 취급하고 복구 또는 마이그레이션 과정에서 보존해야 합니다.

## 스토어 패키징 참고

Microsoft Store 패키징은 portable 릴리스와 다른 쓰기 가능 런타임 루트가 필요할 수 있습니다.

portable 릴리스는 앱 폴더를 런타임 루트로 계속 사용할 수 있습니다.<br>
Store/MSIX 빌드는 설치 패키지 위치를 일반적인 쓰기 가능 런타임 디렉터리로 사용할 수 없으므로,<br>
사용자 쓰기 가능 앱 데이터 위치 또는 사용자가 명시적으로 선택한 작업 공간 폴더를 사용해야 합니다.
