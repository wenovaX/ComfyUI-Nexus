[h]Nexus란 무엇인가요[/h]

Nexus는 ComfyUI의 설치, 실행, 환경 설정, 업데이트를 자동으로 관리해 주는 통합 도구입니다.
당신이 완벽한 환경에서 작업할 수 있도록 파견된 [b]'베테랑 작업실 관리인'[/b]이라 생각해보세요.

[accent]ComfyUI가 아주 강력하지만 다루기 까다로운 기계라면,
Nexus는 그 모든 복잡한 뒷일들을 대신 처리해주는 든든한 가이드 역할을 합니다.[/accent]

---

[h]동작 원리: "마법 작업실 깨우기"[/h]

AI 예술을 시작하기 위해, Nexus는 다음과 같은 순서로 작업실의 불을 밝힙니다.

[b]1. Nexus (스마트 에이전트)[/b]

가장 먼저 움직이는 핵심 요소입니다.
Nexus는 ComfyUI에 미리 파견된 [b]'에이전트'[/b]처럼 방을 정돈하고, 도구를 점검하며,
당신이 쾌적하게 작업할 수 있는 환경을 만듭니다.

[b]2. Python (작업실의 기본 뼈대)[/b]

ComfyUI는 기본적으로 Python이라는 프로그래밍 언어를 기반으로 동작합니다.
Nexus는 사용자가 복잡한 코드를 몰라도 이 Python 환경을 자동으로 구성해 줍니다.

[b]3. WebView (작업창 그 자체)[/b]

사용자는 복잡한 웹 브라우저 설정이나 번거로운 추가 설치 과정 없이,
마치 일반 프로그램처럼 ComfyUI 화면을 띄우고 바로 창작에 들어갈 수 있습니다.

[b]4. Git (유용한 도구 보급로)[/b]

ComfyUI 본체는 물론, 전 세계 개발자들이 만든 수많은 추가 기능(커스텀 노드)을
내 작업실로 안전하게 가져오는 데 사용되는 [b]'공식 통로'[/b]입니다.

---

[h]무엇이 필요한가요 (그리고 왜 필요한가요?)[/h]

[h]1. Python (파이썬)[/h]

이것은 [b]'핵심 엔진'[/b]입니다. 자동차가 움직이기 위해 엔진이 필요하듯,
ComfyUI가 생각하고 이미지를 만들기 위해서는 파이썬이 반드시 필요합니다.

[h]2. 가상 환경 (venv)[/h]

이것은 [b]'전용 도구함'[/b]입니다. 도구들을 바닥에 흩어놓는 대신 정리된 상자에 담아두는 것과 같습니다.
이렇게 하면 컴퓨터가 지저분해지지 않고 다른 프로그램들과 충돌하는 것도 방지할 수 있습니다.
[img:python_environment.png|800]

[h]3. PyTorch & CUDA[/h]

이것은 당신의 [b]'초고속 파워'[/b]입니다.

→ [b]PyTorch[/b]는 AI가 실제로 계산을 수행하는 핵심 도구입니다.
→ [b]CUDA[/b]는 그래픽카드의 힘을 빌려 생성 속도를 크게 향상시켜 줍니다.

[accent]이게 없다면 당신의 AI는 달팽이처럼 느려질 거예요![/accent]

[b]4. Git (깃)[/b]

이것은 [b]'만능 연결 고리'[/b]입니다. 전 세계 창작자들이 만든 최신 기능과
새로운 노드들을 다운로드할 때 사용됩니다.

골치 아픈 설정은 Nexus에게 맡기고, 당신은 오직 AI 창작에만 집중하세요.[/accent]
