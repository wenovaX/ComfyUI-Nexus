[h]Basic Arsenal and Final Check[/h]

First, let's explore the trusty basic arsenal (custom nodes) pre-configured by ComfyUI-Nexus, 
and then complete your final check before hitting the 'Queue' button—just like a pilot inspecting the instruments before takeoff.

---

[h]Your Pre-installed Trusty Arsenal[/h]

The true power of ComfyUI comes from 'custom nodes' (extensions). 
Here are the essential packages pre-installed and curated by Nexus. 
They are ready to use out of the box with no extra setup required!

[b]1. ComfyUI Manager (Your All-in-One Assistant)[/b]  [link:🔗 GitHub|https://github.com/ltdrdata/ComfyUI-Manager]
→ A versatile tool that helps you install or update new nodes and search for and download missing models. 
→ This will be your most frequently used tool and is seamlessly managed and integrated within Nexus. 
→ Use the Manager to easily explore more diverse and exciting custom features and embark on your own adventure!

[b]2. ComfyUI HUD (Head-Up Display)[/b]
→ An innovative convenience feature that extracts core control panels 
from the chaotic, messy noodle soup of nodes and beautifully floats them on your screen. (Nexus Built-in Exclusive)

[b]3. Impact Pack & Subpack (Detailer & Face Correction)[/b]  [link:🔗 GitHub|https://github.com/ltdrdata/ComfyUI-Impact-Pack]
→ An essential suite of tools that automates workflows and boosts quality, 
including the Face Detailer which beautifully restores distorted faces in high resolution.

[b]4. IP-Adapter Plus (Image Hint Injection)[/b]  [link:🔗 GitHub|https://github.com/cubiq/ComfyUI_IPAdapter_plus]
→ A powerful tool that lets you input a 'reference image' instead of a text prompt 
to naturally reflect the composition, style, or subject's features.

[b]5. ControlNet Aux (ControlNet Preprocessors)[/b]  [link:🔗 GitHub|https://github.com/Fannovel16/comfyui_controlnet_aux]
→ A collection of preprocessor nodes used to extract poses (Openpose), depth (Depth), or lines (Canny).

[b]6. AnimateDiff Evolved (Smooth Animations)[/b]  [link:🔗 GitHub|https://github.com/Kosinkadink/ComfyUI-AnimateDiff-Evolved]
→ A package that goes beyond static images to create natural, 
vibrant moving animations (GIF/MP4) to bring your ideas to life.

[b]7. Other Tools (Face Analysis & Prompt Utilities)[/b]
→ [b]FaceAnalysis[/b] [link:GitHub|https://github.com/chflame163/ComfyUI_FaceAnalysis]
→ [b]PromptUtilities[/b] [link:GitHub|https://github.com/Moussekateer/ComfyUI-PromptUtilities]
→ [b]AlekPet Nodes[/b] [link:GitHub|https://github.com/AlekPet/ComfyUI_Custom_Nodes_AlekPet]

are also installed to help you write prompts and analyze faces with ease.

[h]Essential Checklist Before Takeoff[/h]

A single setting can dramatically affect your generation results and wait times. 
Check these three things before queueing!

1. [b]Verify Model Loading (Checkpoint)[/b]
→ Make sure the model matching your intended art style (SD 1.5 / SDXL) is selected correctly. 
→ If a node is glowing red, it indicates a path configuration error, so please re-select the file.

2. [b]Resolution and Aspect Ratio (Latent Image)[/b]
→ Is the resolution appropriate for the model base? (512 is recommended for SD1.5, 1024 for SDXL)
→ Blindly increasing the resolution can cause the application to halt due to an Out of Memory (OOM) error on your graphics card.

3. [b]Sampler and Steps[/b]
→ If Steps are too low, the image details will look muddy. (At least 20 steps recommended)
→ If the CFG scale is too high, the image can look burned or colors can break. (5 to 8 is generally recommended)

[h]How to Deal with Errors Without Panicking[/h]

→ [b]A node turned red![/b]
This happens when you load a workflow containing custom nodes that are not installed or if there is a wrong path. 

Open [b]ComfyUI Manager[/b] and click 'Install Missing Custom Nodes' 
to automatically find and install what you need instantly!

→ [b]I pressed queue, but nothing is happening or it's stuck![/b]
Loading a model for the first time takes some time as it loads into the graphics card memory. 

If it seems completely frozen, check the Console log at the bottom 
to see if there is an Out of Memory message.
