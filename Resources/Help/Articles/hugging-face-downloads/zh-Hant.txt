[h]推薦下載方法[/h]

[img:huggingface_downloand_guide.png|360]

1. 打開模型頁面
2. 導覽至「Files and versions」標籤頁
3. 尋找所需的檔案（.safetensors / .bin）
4. 點擊檔案右側的下載圖示

[accent]請點擊右側的下載圖示 (↓)，而非檔案名稱。[/accent]

---

[h]尋寶實戰技巧[/h]

→ 下載前，請務必確認檔案副檔名（.safetensors、.bin 等）。
  - 通常，.safetensors 檔案在安全性上要可靠得多。
→ 即使是同一個模型，也可能存在多個版本（fp16、fp32 等）。
  - 如果想節省 VRAM，請選擇 fp16 版本；如果需要原始品質，請選擇 fp32。

[h]進階使用者專用（Git / CLI）[/h]

如果你熟悉 Git，可以直接複製整個倉庫，而無需逐一下載檔案。
這在你需要完整設定（模型 + 設定檔等）時非常有用，可以省去挨個下載檔案的麻煩。

[h]使用 Git[/h]

[code]git lfs install
git clone https://huggingface.co/<repo>[/code]

- 完整複製包含大檔案的整個倉庫。
- 最適合一次性下載大量檔案或一整套模型。

[h]使用 Hugging Face CLI[/h]

[code]huggingface-cli download[/code]

- 透過指令下載特定檔案或整個倉庫。
- 比 Git 更靈活，可以挑選下載，非常適合自動化任務。

---

[h]何時使用哪種方法？[/h]

[b]使用 Web UI[/b]
- 下載單一檔案時

[b]使用 Git / CLI[/b]
- 需要下載多個檔案時
- 一次性下載大量模型或套裝時
- 建立可重複配置的環境時
