[h]Recommended Download Method[/h]

[img:huggingface_downloand_guide.png|360]

1. Open the model page
2. Navigate to the "Files and versions" tab
3. Search for the required file (.safetensors / .bin)
4. Click the download icon on the right side of the file

[accent]Click the download icon (↓) on the right, not the file name.[/accent]

---

[h]Treasure Hunting Practical Tips[/h]

→ Before downloading, always check the file extension (.safetensors, .bin, etc.).
  - Usually, .safetensors files are much safer from a security standpoint.
→ Even for the same model, multiple versions (fp16, fp32, etc.) may exist.
  - Choose the fp16 version to save VRAM, or fp32 if you need the original quality.

[h]For Advanced Users (Git / CLI)[/h]

If you are familiar with Git, you can clone the entire repository instead of downloading files one by one.
This is useful when you need the complete setup (models + config files, etc.) and saves you the trouble of downloading files individually.

[h]Using Git[/h]

[code]git lfs install
git clone https://huggingface.co/<repo>[/code]

- Clones the entire repository, including all large files.
- Best when downloading massive amounts of files or a full set of models at once.

[h]Using Hugging Face CLI[/h]

[code]huggingface-cli download[/code]

- Downloads specific files or repositories via commands.
- More flexible than Git for selective downloading and highly useful for automation tasks.

---

[h]When to Use What?[/h]

[b]Using Web UI[/b]
- When downloading a single file

[b]Using Git / CLI[/b]
- When multiple files are needed
- When downloading a large batch of models or a set all at once
- When configuring a reproducible environment
