[h]推荐下载方法[/h]

[img:huggingface_downloand_guide.png|360]

1. 打开模型页面
2. 导航至“Files and versions”标签页
3. 查找所需的文件（.safetensors / .bin）
4. 点击文件右侧的下载图标

[accent]请点击右侧的下载图标 (↓)，而非文件名。[/accent]

---

[h]寻宝实战技巧[/h]

→ 下载前，请务必确认文件扩展名（.safetensors、.bin 等）。
  - 通常，.safetensors 文件在安全性上要可靠得多。
→ 即使是同一个模型，也可能存在多个版本（fp16、fp32 等）。
  - 如果想节省 VRAM，请选择 fp16 版本；如果需要原始质量，请选择 fp32。

[h]高级用户专用（Git / CLI）[/h]

如果你熟悉 Git，可以直接克隆整个仓库，而无需逐个下载文件。
这在你需要完整设置（模型 + 配置文件等）时非常有用，可以省去挨个下载文件的麻烦。

[h]使用 Git[/h]

[code]git lfs install
git clone https://huggingface.co/<repo>[/code]

- 完整克隆包含大文件的整个仓库。
- 最适合一次性下载大量文件或一整套模型。

[h]使用 Hugging Face CLI[/h]

[code]huggingface-cli download[/code]

- 通过命令下载特定文件或整个仓库。
- 比 Git 更灵活，可以挑选下载，非常适合自动化任务。

---

[h]何时使用哪种方法？[/h]

[b]使用 Web UI[/b]
- 下载单个文件时

[b]使用 Git / CLI[/b]
- 需要下载多个文件时
- 一次性下载大量模型或套装时
- 构建可重复配置的环境时
