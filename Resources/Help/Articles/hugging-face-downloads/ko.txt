[h]권장 다운로드 방법[/h]

[img:huggingface_downloand_guide.png|360]

1. 모델 페이지 열기
2. "Files and versions" 탭으로 이동
3. 필요한 파일 탐색 (.safetensors / .bin)
4. 파일 우측의 다운로드 아이콘 클릭

[accent]파일 이름이 아닌 우측의 다운로드 아이콘(↓)을 클릭하세요.[/accent]

---

[h]보물찾기 실전 팁[/h]

→ 다운로드 전, 파일 확장자(.safetensors, .bin 등)를 꼭 확인하세요.
  - 보통 .safetensors 파일이 보안상 훨씬 안전합니다.
→ 같은 모델이라도 여러 버전(fp16, fp32 등)이 존재할 수 있습니다.
  - VRAM을 아끼려면 fp16 버전을, 원본 퀄리티가 필요하면 fp32를 선택하세요.

[h]고급 사용자용 (Git / CLI)[/h]

Git에 익숙하다면 파일을 하나씩 받는 대신 전체 저장소를 복제할 수 있습니다.
전체 설정(모델 + 설정 파일 등)이 모두 필요할 때 유용하며, 파일을 일일이 받는 수고를 덜어줍니다.

[h]Git 사용하기[/h]

[code]git lfs install
git clone https://huggingface.co/<repo>[/code]

- 대용량 파일이 포함된 저장소 전체를 통째로 복제합니다.
- 수많은 파일이나 모델 세트를 한 번에 다운로드할 때 가장 좋습니다.

[h]Hugging Face CLI 사용하기[/h]

[code]huggingface-cli download[/code]

- 명령어를 통해 특정 파일이나 전체 저장소를 다운로드합니다.
- Git보다 유연하게 골라 받을 수 있고 자동화 작업에 유용합니다.
