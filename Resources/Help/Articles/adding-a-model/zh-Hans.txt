[h]需要模型时（去哪里找？）[/h]

如果你是新手，请先从 [b]Civitai[/b] 开始。  [article:了解 Civitai|civitai]

[h]下载时只需确认这些[/h]

在模型详情页面检查以下三点，就能避免大多数失误。

1. [b]确认 Base Model 版本[/b]
→ 确认是 [b]SD 1.5[/b] 还是 [b]SDXL[/b]。
→ 之后使用的 LoRA 或 ControlNet 也必须匹配此版本。

2. [b]查看制作者说明[/b]
→ 可能会列出推荐 VAE、触发词、推荐分辨率和采样器设置。
→ 需要单独 VAE 的模型通常会在说明或下载信息中注明。

3. [b]整理文件名[/b]
→ 下载时请将文件名整理得直观易懂。
→ 例如：sdxl_real_modelname_v1.safetensors / sd15_anime_modelname.safetensors
→ 区分 SD1.5 / SDXL / 风格 / 版本，日后查找时会大大节省时间。
