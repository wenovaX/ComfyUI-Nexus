[h]다운로드할 때 이것만 확인하세요[/h]

모델 상세 페이지에서 아래 세 가지만 체크해도 대부분의 실수를 피할 수 있습니다.

1. [b]Base Model 버전 확인[/b]  
→ [b]SD 1.5[/b]인지 [b]SDXL[/b]인지 반드시 확인하세요.  
→ 이후 사용할 LoRA나 ControlNet도 이 버전에 맞춰야 합니다.

2. [b]제작자 설명 확인[/b]  
→ 권장 VAE, Trigger words, 권장 해상도, 샘플러 설정이 적혀 있을 수 있습니다.  
→ 특히 VAE를 따로 써야 하는 모델은 보통 설명이나 다운로드 정보에 안내되어 있습니다.

3. [b]파일명 정리[/b]  
→ 다운로드할 때 파일명을 직관적으로 정리해 두세요.  
→ 예: sdxl_real_model이름_v1.safetensors / sd15_anime_model이름.safetensors
→ SD1.5 / SDXL / 스타일 / 버전을 구분해 두면 나중에 찾는 시간이 크게 줄어듭니다.
