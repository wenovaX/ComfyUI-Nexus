[h]Check These Before Downloading[/h]

Just checking these three things on the model detail page will help you avoid most mistakes.

1. [b]Verify the Base Model Version[/b]
→ Make sure it's either [b]SD 1.5[/b] or [b]SDXL[/b].
→ Any LoRA or ControlNet you use later must match this version.

2. [b]Read the Creator's Description[/b]
→ Recommended VAE, trigger words, recommended resolution, and sampler settings may be listed.
→ Models that require a separate VAE usually mention it in the description or download info.

3. [b]Organize Your Filenames[/b]
→ Rename files intuitively when downloading.
→ Example: sdxl_real_modelname_v1.safetensors / sd15_anime_modelname.safetensors
→ Distinguishing SD1.5 / SDXL / style / version saves a lot of time searching later.
