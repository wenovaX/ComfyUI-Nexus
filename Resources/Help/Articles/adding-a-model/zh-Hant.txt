[h]需要模型時（去哪裡找？）[/h]

如果你是新手，請先從 [b]Civitai[/b] 開始。  [article:了解 Civitai|civitai]
可以像逛商場一樣瀏覽預覽圖來挑選模型，是最直覺的方式。

熟悉之後，也可以逛逛 [b]Hugging Face[/b]。  [article:了解 Hugging Face|hugging-face-downloads]
那是一個像 Costco 一樣的巨大寶庫，可以在那裡發掘尚未上架市場的原石。

[h]下載時只需確認這些[/h]

在模型詳情頁面檢查以下三點，就能避免大多數失誤。

1. [b]確認 Base Model 版本[/b]
→ 確認是 [b]SD 1.5[/b] 還是 [b]SDXL[/b]。
→ 之後使用的 LoRA 或 ControlNet 也必須配合此版本。

2. [b]查看製作者說明[/b]
→ 可能會列出推薦 VAE、觸發詞、推薦解析度和取樣器設定。
→ 需要單獨 VAE 的模型通常會在說明或下載資訊中標註。

3. [b]整理檔案名稱[/b]
→ 下載時請將檔案名稱整理得直觀易懂。
→ 例如：sdxl_real_modelname_v1.safetensors / sd15_anime_modelname.safetensors
→ 區分 SD1.5 / SDXL / 風格 / 版本，日後搜尋時會大大節省時間。
