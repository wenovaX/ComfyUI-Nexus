[h]고급 퀘스트: 주인공의 얼굴 고정하기[/h]

생성할 때마다 주인공의 얼굴이 계속 바뀌어 고민인가요?
특정 인물의 얼굴을 모든 이미지에 일관되게 유지하는 기술을 **'페이스 싱크(Face Sync)'**라고 합니다.

[accent]🔥 난이도 선언: 이 퀘스트는 추가 라이브러리 설치와 전용 모델 수집이 필수적인 [b]'졸업급 심화 퀘스트'[/b]입니다. 
기본 이미지 생성에 충분히 익숙해진 뒤, 여유롭게 이 모험을 시작해 보세요![/accent]

[h]AI에게 길 묻기[/h]

아래 프롬프트를 복사해서 GPT, Gemini, Copilot 등 원하는 AI에 붙여넣기해 보세요.
모델 다운로드부터 노드 연결까지, 완전한 초보자도 따라할 수 있는 안내를 받을 수 있습니다.

[copy:AI 프롬프트 복사하기|아래 프롬프트를 GPT, Gemini, Copilot 등 원하는 AI에 붙여넣기하세요.|프롬프트를 클립보드에 복사합니다.|복사됨|클립보드에 복사되었습니다.]
ComfyUI에서 IP-Adapter FaceID를 사용하여 특정 인물의 얼굴을 고정하는 워크플로우를 만들고 싶습니다.

완전한 초보자가 따라할 수 있도록 아래 내용을 순서대로 알려주세요:

1. 필요한 모델 파일 목록과 각각의 다운로드 링크 (CLIP Vision, IP-Adapter FaceID 가중치, InsightFace buffalo_l)
2. 각 모델 파일을 ComfyUI의 어느 폴더에 넣어야 하는지
3. ComfyUI에서 노드를 꺼내는 순서 (Load Checkpoint부터 Save Image까지)
4. 각 노드의 출력을 어디에 연결하는지 (슬롯 이름 포함)
5. IP-Adapter 노드의 weight 값 권장 범위
6. 참조 얼굴 사진의 권장 크기와 크롭 팁
7. FaceDetailer 노드를 마지막에 추가하여 얼굴 디테일을 보정하는 방법

내가 사용 중인 베이스 모델: [여기에 SD 1.5 또는 SDXL 입력]
[/copy]

[h]이미 지급된 기본 무기[/h]

고맙게도 IP-Adapter 노드는 Nexus의 [b]기본 패키지[/b]에 이미 포함되어 함께 빌드되어 있습니다. 
따라서 별도의 확장 노드를 설치하느라 시간 낭비할 필요가 없습니다!

워크플로우 편집 창에서 마우스 우클릭 후 'IPAdapter'를 검색해 보세요. 
어떤 단자들을 와이어링해야 이 강력한 노드가 잠깨어 동작하는지, 
스스로 고민하고 부딪히며 해답을 완성해 나가는 짜릿함을 느껴보세요.

→ [link:🔗 IP-Adapter 공식 GitHub (참고용)|https://github.com/cubiq/ComfyUI_IPAdapter_plus]

[h]모험가의 무기 관리 도구: ComfyUI-Manager[/h]

비록 이번 노드는 Nexus가 미리 챙겨주었지만, 
앞으로 떠날 수많은 심화 모험에서는 새로운 커스텀 노드를 수집하고 버전을 관리해야 할 순간이 반드시 찾아옵니다.

워크플로우 화면 우측 하단의 [b]'Manager'[/b] 버튼을 눌러보세요. 
이 강력한 중앙 통제기를 통해 필요한 확장 노드를 직접 검색해 보고, 
누락된 종속성을 스스로 관리하는 방식을 체득하는 것은 
일류 모험가(Senior Builder)로 거듭나기 위한 최고의 필수 관문입니다.

[h]탐험에 필요한 3가지 필수 재료[/h]

이 마법을 직접 구축해내려면, 아래의 3가지 핵심 재료들을 올바른 경로에 수집해 두어야 합니다.

[b]1. CLIP Vision 모델 (AI의 제2의 눈)[/b]
→ [b]수집품:[/b] SD 1.5용인 [b]ViT-H[/b]와 SDXL용인 [b]ViT-bigG[/b] 중, 
내가 사용 중인 베이스 모델에 맞는 버전을 탐색하여 
[folder:ComfyUI/models/clip_vision|models/clip_vision] 경로에 배치해야 합니다.
→ [link:🔗 CLIP Vision 탐색 출발지|https://huggingface.co/h94/IP-Adapter/tree/main/models]

[b]2. IP-Adapter FaceID 가중치 (Weights/Presets)[/b]
→ [b]수집품:[/b] 실제 인물의 얼굴 형태를 반영하는 프리셋 파일입니다. 
마찬가지로 SD 1.5용과 SDXL용이 나뉘어 있습니다.
→ [b]경로:[/b] [folder:ComfyUI/models/ipadapter|models/ipadapter] 경로에 배치해야 합니다.
→ [b]탐색 힌트:[/b] 일부 커스텀 로더 노드가 특정 확장자(예: [b]`.bin` 확장자[/b])만을 편식하는 
유서 깊은 미스터리가 있으니 수집 시 주의가 필요합니다.
→ [link:🔗 FaceID 전용 가중치 탐색 출발지|https://huggingface.co/h94/IP-Adapter-FaceID/tree/main]

[b]3. InsightFace 분석 모델 (buffalo_l)[/b]
→ [b]수집품:[/b] 소스 사진에서 얼굴의 윤곽선과 오관의 정확한 좌표를 정밀 측정해내는 보이지 않는 탐지기 엔진입니다. 
검출 정확도가 가장 뛰어난 [b]'buffalo_l'[/b] 모델이 표준으로 쓰이며, 
최초 실행 시 보통 백그라운드에서 자동 빌드 및 다운로드됩니다.
